# NayePankh AI Help Desk

## Overview

NayePankh AI Help Desk is a modern NGO support platform built using React, Vite, Tailwind CSS, React Router, and Gemini API.

The platform helps users explore volunteering opportunities, internships, donations, and interact with an AI-powered support assistant through a clean and responsive interface.

## Live Demo

https://naye-pankh-ai-help-desk.vercel.app/

## Inspiration Behind the Project

The idea for this project came from observing how many NGOs struggle to provide organized digital platforms for volunteers, interns, and donors. Most small organizations either have outdated websites or lack an interactive support system that helps users quickly understand how they can contribute.

I wanted to create a platform that feels modern, trustworthy, and easy to use while also solving real communication problems for NGOs. The AI Help Desk concept was introduced to make the experience more interactive and accessible for users who may have questions about volunteering, internships, donations, or contacting the organization.

The goal was to combine social impact with modern frontend technology and create something that looks and feels like a real-world startup or NGO product rather than a basic college assignment.

## How This Project Helps

This platform can help NGOs by:

* Making volunteering opportunities easier to discover
* Helping students apply for internships in a simple way
* Encouraging donations through clear information
* Providing instant responses through AI support
* Improving communication between NGOs and the community
* Creating a more professional online presence for organizations

The AI Help Desk improves accessibility by allowing users to quickly ask questions instead of searching manually through different sections of the website.

## Problems I Faced and Solved

### GitHub Push and Merge Conflicts

I faced issues while pushing the project to GitHub due to merge conflicts in the README file. I learned how to resolve conflicts manually using Git commands.

### Gemini API Errors

I encountered API issues such as:

* 404 model errors
* 429 quota exceeded errors

I understood how API keys, quotas, and environment variables work in React and Vercel deployment.

### Dark Mode Issues

Dark mode initially did not work properly because of Tailwind CSS configuration changes. I learned how to implement dark mode using React state, localStorage, and Tailwind custom variants.

### Responsive Navbar Issues

The mobile navbar was showing incorrectly because of browser width and responsive Tailwind classes. I learned how responsive breakpoints work in Tailwind CSS.
